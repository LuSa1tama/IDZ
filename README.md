# 🏠 Прогнозирование цены недвижимости

Это проект по машинному обучению для прогнозирования стоимости квартир на основе различных характеристик.

## Описание проекта
Данный проект посвящен прогнозированию стоимости недвижимости с использованием методов машинного обучения. 

Цель проекта — построить модель, способную предсказывать цену объекта недвижимости на основе его характеристик.

**Задача:** Регрессия — предсказание цены недвижимости.

**Данные:**
- Площадь квартиры (м²)
- Количество комнат
- Расстояние до центра (км)
- Уровень инфраструктуры (1-10)
- Район города

## Использованные модели

- Linear Regression (лучшая модель)
- Random Forest Regressor
- Gradient Boosting Regressor
- Decision Tree Regressor

## Структура проекта

- `app.py` — веб-приложение на Streamlit
- `best_model.pkl` — обученная модель (Linear Regression)
- `requirements.txt` — необходимые библиотеки
- `идз (5).py` / `.ipynb` — ноутбук с анализом данных и обучением моделей
- `real_estate_dataset.csv` — датасет

## Как запустить локально

pip install -r requirements.txt
